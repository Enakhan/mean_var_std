# Numpy Stats Calculator & Dashboard 📊

This is a Python-based data analysis tool that calculates statistical metrics from a list of numbers and visualizes the results through an interactive dashboard.

## 🚀 Project Overview
Developed as part of my journey toward becoming a **Machine Learning Engineer**, this project demonstrates my ability to handle matrix operations and create professional data visualizations. It is a key addition to my portfolio for **Data Science and AI** master's applications in Europe.

## ✨ Key Features
*   **Matrix Transformation**: Converts a 9-element input list into a $3 \times 3$ NumPy matrix.
*   **Statistical Analysis**: Provides Mean, Variance, Standard Deviation, Max, Min, and Sum for rows, columns, and the entire matrix.
*   **Visual Dashboard**: Includes a multi-chart visual report using **Matplotlib**.
*   **Unit Testing**: Built to pass automated test cases for reliability.

## 🛠️ Technologies Used
*   **Python**: Core logic and scripting.
*   **NumPy**: High-performance matrix calculations.
*   **Matplotlib**: Dynamic data visualization and dashboarding.
*   **Replit**: Integrated development and hosting environment.

## 📁 File Structure
*   `mean_var_std.py`: Contains the `calculate()` function with NumPy logic.
*   `main.py`: Entry point for running the application and tests.
*   `test_module.py`: Automated tests to ensure accuracy.

## 💡 How to Run on Replit
1. Click the **Run** button at the top of the Replit interface.
2. The program will execute `main.py` and display the statistical results in the console.
3. Any generated plots will appear in the Replit "Files" or "Output" section.
